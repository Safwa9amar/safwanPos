# **App Name**: PrismaPOS

## Core Features:

- Product Lookup: Scan a barcode to retrieve product details from the MySQL database.
- Real-time Cart Management: Dynamically update the cart with product details, quantities, and calculated line totals.
- Transaction Processing: Record sales transactions in the MySQL database.
- Sales Reporting Tool: Use an AI tool to aggregate and summarize the current transactions, and use this summary for deciding which products may be out of stock.
- Minimalist UI: A clean, intuitive two-column layout for efficient transaction processing.
- Authentication: Secure the POS system using Firebase Authentication.
- Facture Generation: Generate a simple, printable receipt after each successful transaction.

## Style Guidelines:

- Primary color: Soft blue (#90AFC5) to create a calm and trustworthy atmosphere.
- Background color: Light gray (#EAEAEA) for a clean and neutral canvas.
- Accent color: Muted orange (#D9534F) for call-to-action elements and important alerts.
- Body and headline font: 'Inter', a sans-serif for a modern, machined, objective, neutral look; appropriate for both headlines and body text.
- Simple, consistent icons for product categories and actions.
- Two-column layout with the main POS area on the left and the cart summary on the right.
- Subtle transitions and feedback animations when adding or removing items from the cart.
